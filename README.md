# Typing Dojo — ブラインドタッチ練習アプリ

シンプルで使いやすいブラインドタッチ（タッチタイピング）習得アプリです。  
インストール不要・外部依存なし。`index.html` をブラウザで開くだけで使えます。

## 機能

- **5つのレッスン** — ホームポジション / 上段 / 下段 / 単語 / 文章
- **リアルタイム統計** — WPM（1分間の打鍵ワード数）・精度・経過時間
- **カラーコード付きキーボード図** — 指ごとに色分けして正しい運指を視覚化
- **次打ちガイド** — 次に押すキーと使う指を常に表示
- **完了サマリー** — レッスン終了時にスコアをモーダルで確認

## 指の色分け

| 色 | 指 |
|----|----|
| 🔴 赤 | 小指 |
| 🟠 オレンジ | 薬指 |
| 🟡 黄 | 中指 |
| 🟢 緑 | 人差し指 |
| 🔵 青 | 親指（スペース）|

## 使い方

1. `index.html` をブラウザで開く
2. 画面をクリックして練習スタート
3. 表示されたテキストをキーボードで入力
4. キーボード図のハイライトと「次のキー」ガイドを参考に、キーボードを見ずに打つ

## 推奨練習順序

1. **ホームポジション** — `asdf jkl;` の基本姿勢を体で覚える
2. **上段** — `qwert yuiop` に指を伸ばす練習
3. **下段** — `zxcv bnm` の下段キー
4. **単語** — 実際の英単語でリズムを養う
5. **文章** — スムーズなタイピングへ
